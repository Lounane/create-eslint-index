> Simplify the creation an index file for your ESLint plugin


## Install

```
$ npm install --save create-eslint-index
```


## Usage

```js
const createEslintIndex = require('create-eslint-index');

createEslintIndex('unicorns');
//=> 'unicorns & rainbows'
```
